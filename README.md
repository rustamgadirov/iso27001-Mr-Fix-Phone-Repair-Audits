# ISO 27001 – Physical Security (Annex A.7) Audit Project

This repository contains a sample ISO 27001 Annex A.7 physical security audit for a small phone repair business ("Mr Fix Phone Repair").

## What this project shows

- Understanding of ISO 27001 Annex A.7 (Physical and Environmental Security)
- Ability to identify nonconformities in physical security controls
- Writing professional audit findings:
  - Requirement (ISO reference)
  - Issue / Nonconformity
  - Evidence
  - Impact (risk)
  - Recommendation (corrective action)
